Este projeto foi feito dentro do modulo de aprofundamento dos hooks no curso de <strong>Reactjs</strong> da <strong>RocketSeat</strong>; <br> 

Este projeto foi criado com o <strong>Vite</strong>, desenvolvido com <strong>React+Typescript</strong>, <strong>Styled-components</strong> para estilização, <strong>React-Route-dom</strong> para as rotas entre paginas,  
<strong>React Hook form</strong> para trabalhar com formulários e <strong>Zod</strong> para validação, para a logica do projeto foi utilizado contexto, reducer com o <strong>Immer</strong>;<br> 

Este projeto é um tipo de temporizador, onde colocamos o nome da tarefa, tempo que queremos disponibilizar para tarefa escolhida, podemos esperar o tempo encerrar ou podemos interromper,  
além da pagina que mostra o relógio tem a pagina de histórico, no histórico podemos ver a lista de tarefas que foi executados mostrando seus status(concluído, interrompido e em andamento);<br> 

para executar: ```npm i``` e depois ```npm run dev``` 

-------------------------------------------------------------------------------------------------------------------------

This project was done within the hooks in-depth module in the <strong>Reactjs</strong> course at <strong>RocketSeat</strong>; <br>

This project was created with <strong>Vite</strong>, developed with <strong>React+Typescript</strong>, <strong>Styled-components</strong> for styling, <strong>React-Route-dom< /strong> for routes between pages,
<strong>React Hook form</strong> to work with forms and <strong>Zod</strong> for validation, for the project logic we used context, reducer with <strong>Immer</strong>;<br>

This project is a type of timer, where we enter the name of the task, the time we want to make available for the chosen task, we can wait for the time to end or we can interrupt,
in addition to the page that shows the clock there is a history page, in the history we can see the list of tasks that were performed showing their status (completed, interrupted and in progress);<br>

to run: ```npm i``` and then ```npm run dev```
